3to1go Edge

Filesystem-driven backup selection

Browse the scan root, inspect current pathing, and create or update .upload_dir markers without touching the terminal. Edge scans from the configured root, aggregates matching job directories, and uploads changed archives to Central.

Selected Jobs -

Existing .upload_dir markers already appear here automatically. Use Edit to review or change upload settings for a folder. If you need to stop backing up that folder, do it from inside the job editor.

Directories Under Scan Root

Click Edit on an available directory to create or update its .upload_dir settings. Nested folders under an already-selected parent are covered by that parent job and do not open their own job editor here.
